# JuegoDVD

## Especificación del proyecto
- El proyecto se ha desarrollado con Java 17.
- El proyecto utiliza el framework java swing.

## ¿Que es este proyecto?
Este repositorio contiene un juego que simula el movimiento de un DVD por la pantalla, rebotando en los bordes. El objetivo del juego es hacer click en el boton de DVD que rebota en las esquinas de la pantalla.

Este proyecto se hizo como trabajo de clase de Diseño de Interfaces de 2do grado de Formacion Profesional de Desarrollo de Aplicaciones Multiplataforma.
